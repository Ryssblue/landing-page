# TokoKu Landing Page

TokoKu adalah landing page e-commerce statis modern dengan keranjang belanja, quick view produk, dark mode, filter kategori, pencarian, sorting, toast notifikasi, dan checkout WhatsApp.

## Struktur

```text
tokoku/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   ├── icons/
│   └── products/
└── README.md
```

## Cara Menjalankan

Buka `index.html` langsung di browser, atau jalankan server statis lokal dari folder `tokoku`.

## Konfigurasi WhatsApp

Nomor WhatsApp ada di `script.js`:

```js
const WHATSAPP_NUMBER = "6283125726235;
```

Ganti dengan nomor toko dalam format internasional tanpa tanda `+`.
