Apakah checkout langsung masuk ke WhatsApp?

Ya, tombol checkout akan membuka WhatsApp dengan daftar pesanan dan total harga otomatis.

Apakah keranjang hilang setelah refresh?

Tidak. Keranjang tersimpan di Local Storage browser sampai kamu menghapus itemnya.

Bisakah nomor WhatsApp diganti?

Bisa. Nomor disiapkan di file JavaScript agar mudah disesuaikan untuk toko.
